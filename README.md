# ChatGPT UX Improvements (Prototype)

This project recreates the ChatGPT web UI and layers in UX concept improvements for rapid user testing.

**Planned feature concepts**
- Branch chat from a selected message (split view)
- Per-chat answer pinning
- Parking questions (queue prompts while generating)
- Full-view history for Projects + Chats
- Optional Guardrails panel with preset chips

**Stack**
- React + Vite
- Tailwind CSS

**Run locally**
```bash
npm install
npm run dev
```

**Status**
- Baseline UI replicated from screenshots
- Feature work in progress
